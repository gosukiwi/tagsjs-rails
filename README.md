# Tagsjs::Rails
Easily use [tagsjs](https://github.com/gosukiwi/tagsjs) in your Rails apps!

## Usage
In your `application.js`

```javascript
//= require tagsjs/core
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tagsjs-rails'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install tagsjs-rails
```
